# Game Introduction

本專案旨在開發一款經典的 2D 橫向捲軸平台遊戲，核心機制參考《超級瑪利歐》。玩家將操作角色在具備重力物理的環境中進行跳躍、移動與探索。專案需實作嚴謹的碰撞偵測與角色狀態管理，並透過互動方塊獲取資源。

參考影片：https://www.youtube.com/watch?v=2on4G_hb-_c
